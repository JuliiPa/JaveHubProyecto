# JaveHub

## Descripción del Proyecto
JaveHub es una plataforma web diseñada como una red académica colaborativa exclusiva para estudiantes de la Pontificia Universidad Javeriana. El sistema integra funcionalidades de foros, recursos, interacción estudiantil y navegación guiada mediante una arquitectura modular basada en Next.js.

Este proyecto contiene el frontend completo, organizado para permitir ampliaciones futuras como autenticación, integración con backend, manejo de datos y módulos avanzados. La estructura está diseñada para mantener claridad, escalabilidad y separación lógica de responsabilidades.

## Tecnologías Utilizadas
- Next.js 14
- React
- TypeScript
- JavaScript
- TailwindCSS
- PostCSS
- Node.js (npm o pnpm)
- Arquitectura basada en componentes modulares

## Instalación y Ejecución

### Clonar el repositorio
```bash
git clone https://github.com/JuliiPa/JaveHubProyecto.git
cd JaveHubProyecto
```

## Ejecutar el servidor
- npm run dev

## Abrir en el navegador:
- http://localhost:3000

# Estructura del Proyecto
- /app:
  Rutas principales, páginas y estructura general de navegación.

- /components:
  Componentes reutilizables de UI, secciones, layouts y elementos visuales.

- /hooks:
  Hooks personalizados utilizados en diferentes vistas y lógicas del proyecto.

- /contexts:
  Proveedores globales de contexto para manejo de estado y funciones compartidas.

- /lib:
  Funciones auxiliares, herramientas y utilidades generales.

- /public:
  Archivos estáticos como imágenes, íconos, logos y recursos multimedia.

- /styles:
  Archivos de estilos globales y configuraciones de TailwindCSS.

- next.config.mjs:
  Configuración central de Next.js.

- postcss.config.mjs:
  Configuración de PostCSS.

- tsconfig.json:
  Configuración de TypeScript.

- package.json:
  Lista de dependencias, scripts y metadatos del proyecto.
